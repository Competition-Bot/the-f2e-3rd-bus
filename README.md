## 作品說明

Take a bus ! 全台等公車統合全台的公車資訊，提供公車動態、路線規劃以及最新消息三項服務，我們重新定位公車查詢網站的UX流程，希望打造更符合使用者需求的公車查詢服務。

## 系統說明
本次專案運行方式是使用yarn
1. Clone
2. Run `yarn install`
3. Run `yarn start`
4. Open http://localhost:3000

## 資料夾說明
src/
* actions-功能函式
* api-串接TDX
* assets-存放圖片與icon
* components-元件
* json
* pages-頁面
* reducer-管理全域變數的資料
* utils-定義功能名稱
